# Life Queue Dashboard

A personal life dashboard inspired by Valorant’s UI to track daily habits, salah, fitness, and progress.

## Features
- Queue system for daily tasks (Physical, Deen, etc.)
- Salah tracking with optional Sunnah & Tahajjud
- Fitness tracking (weight, calories, sleep, hydration)
- Progress system with RR-style ranking (win/lose based on completion)
- Career tab showing past “days” like match history
- Weight tracking graph with real data

## How to use
1. Open `index.html`
2. Select a queue (Physical / Deen / etc.)
3. Complete tasks and click “Done”
4. Track your progress in the Career and Progress tabs

## Notes
- Data is currently stored locally in the browser
- Designed for personal use first — improvements coming later
